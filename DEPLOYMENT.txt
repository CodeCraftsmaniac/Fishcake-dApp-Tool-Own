DEPLOYMENT PROCESS:
-------------------
1. Push to GitHub main branch
2. GitHub Actions CI runs (build, lint, test)
3. GitHub Actions SSH deploys to Oracle VM
4. PM2 restarts the backend service
5. Health check passes

TROUBLESHOOTING:
----------------
If backend is down:
1. SSH into VM: ssh -i ~/.ssh/oracle_fcc_bot_v3 opc@129.80.144.145
2. Check PM2: pm2 status
3. Check logs: pm2 logs fishcake-backend
4. Restart: pm2 restart fishcake-backend
5. Check health: curl http://localhost:3001/health

If frontend can't connect:
1. Verify NEXT_PUBLIC_API_URL points to VM
2. Check CORS configuration in .env
3. Test backend health endpoint
4. Check browser console for errors
